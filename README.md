# BoardParty
构建一个用于**线下桌游聚会辅助的小型网站系统**。系统主要用于记录排行榜、统计玩家表现，并提供如随机惩罚这样的娱乐功能。

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
